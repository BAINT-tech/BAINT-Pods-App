# BAINT Pods App (MVP)

Smart wireless earbuds app powered by **BAINT AIOPs**.

## Features
- Sound Equalizer (Bass/Mid/Treble)
- Device Status & Battery Simulation
- **Pods Lock (Anti-Theft UI, firmware coming soon)**
- Secure UI matching BAINT branding
